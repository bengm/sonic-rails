# Sonic Rails
AngularJS + Ruby on Rails app to control Sonic Pi

## Installation
*Windows Users should perform the following steps using the Git Shell*

1. `npm install`
2. `bundle install`
3. `rake bower:install`

## SonicPi
[Sonic Pi](http://sonic-pi.net/) must be installed and running.
